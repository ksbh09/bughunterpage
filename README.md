🐛 Bug Hunter — Personal Portfolio
A dark, moody personal portfolio site for a security researcher / bug bounty hunter. Built with pure HTML & CSS — no frameworks, no dependencies.

📸 Features

🖤 Dark hacker-core aesthetic with scanline & noise overlays
⚡ Glitch animation on hero title
🖱️ Custom animated cursor with glow effect
💻 Interactive terminal block in the About section
📊 Animated skill bars on scroll
🃏 Project cards with severity badges (Critical / High / Medium)
🎞️ Scroll-triggered fade-in animations throughout
📱 Fully responsive for mobile


🧰 Built With

HTML5
CSS3 (custom properties, animations, grid)
Vanilla JavaScript (cursor, scroll animations)
Google Fonts — Bebas Neue, Share Tech Mono, Space Mono
